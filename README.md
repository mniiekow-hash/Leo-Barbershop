# Leo Barbershop

This is the website for Leo Barbershop, built using HTML and CSS and deployed via GitHub Pages.

## How to Edit Your Website
- Edit `index.html` to update your website’s content.
- Your live website will be automatically updated when you push changes.

## Live Site
(After setup, your website will be here)

https://mniiekow-hash.github.io/Leo-Barbershop/

## How to Enable GitHub Pages
In case you need to enable GitHub Pages again:
1. Go to **Settings → Pages** in this repository.
2. Under **Build and deployment → Source**, select `main` and `/ (root)` then **Save**.
3. Wait 1–2 minutes for your site to go live at the link above.
